# Confluence_AI

AI-агент для роботи з Confluence Cloud API.

## Можливості
- Читання сторінок Confluence
- Аналіз вмісту та генерація summary через OpenAI
- Оновлення сторінок через API
- FastAPI ендпоїнти для інтеграції
- Модульна архітектура, готова до розширення (Jira, Telegram тощо)

## Структура проєкту
- `clients/` — клієнти для зовнішніх API (Confluence, OpenAI)
- `services/` — бізнес-логіка (обробка даних, робота з контентом)
- `agents/` — логіка AI-агентів
- `api/` — FastAPI ендпоїнти та роути
- `utils/` — допоміжні функції (логування, парсинг)
- `scripts/` — скрипти для автоматизації та міграцій
- `tests/` — модульні та інтеграційні тести

## Налаштування
1. Скопіюйте `.env.example` у `.env`
2. Заповніть необхідні токени
3. Встановіть залежності: `pip install -r requirements.txt`
4. Запустіть проєкт: `python run.py`
